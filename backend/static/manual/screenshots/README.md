Manual screenshots live in the role and module folders under this directory.

Recommended filename format:
`manual_<role>_<module>_<step_number>_<short_description>.png`

Examples:
- `manual_customer_orders_01_open_list.png`
- `manual_employee_documents_02_verify_file.png`
- `manual_admin_reports_03_filter_period.png`

Folders currently prepared for this project:
- `account/`
- `admin/`
- `customer/`
- `employee/`
- `general/`
- `provider/`
- `reports/`
- `settings/`
